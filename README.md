## What RechCS2 Academy is

RechCS2 Academy is aimed at Counter-Strike 2 players who want a more organized way to assess performance and sharpen decision-making. It brings together coach-style analysis and training references so users can review match patterns, monitor progress, and revisit tactical concepts from a single interface.

The app is intended for both mobile and desktop usage, so it stays practical whether you are checking numbers on the move or studying content on a larger screen. Thanks to IndexedDB-backed local storage and delivery via GitHub Pages, it works as a browser-first solution without needing a separate desktop install.

---

## Core features

- Mobile and desktop hubs for convenient use on different devices
- Match analysis centered on how performance changes over time
- Statistics screens that support coaching and self-review
- Utility and nade reference library for fast in-game lookup
- Map-based tactical material for planning and preparation
- Seeded nade entries to help build out the training library
- Offline-friendly browser storage with IndexedDB
- Web hosting through GitHub Pages

## How to use it

Open the web app first, then choose the area that fits what you want to analyze or train.

- Use the match analysis section to inspect performance trends and compare sessions.
- Check statistics to follow progress across games or training cycles.
- Browse the nade reference database when preparing utility usage for a map.
- Review tactical content to plan round ideas and map-specific approaches.
- Return to the platform over time to observe how tracked performance changes.

Because the app keeps data in the browser, the same device and browser profile should be used if you want to preserve local records.

## Configuration

RechCS2 Academy keeps local information in the browser through IndexedDB. For standard use, no separate desktop configuration file is needed.

Settings and stored content are handled inside the web app. If you need to clear local data, remove the site data from your browser settings.
